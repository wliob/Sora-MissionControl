# Kanban Board

This directory holds the project's Kanban board state. It is managed by
the n8n automation workflows and read by department leads for task tracking.

## Columns

- **backlog/** — Tasks not yet scheduled
- **todo/** — Tasks ready to be picked up
- **in-progress/** — Tasks currently being worked on
- **review/** — Tasks awaiting review
- **done/** — Completed tasks
